# WerewolfGamePlugin
マイクラ内で人狼ゲームをするプラグイン。ただし検証に不完全な部分があります。

## コマンド
### 要OP
- /wjoin (player) : 人狼ゲームへの参加登録
- /wleave (player) : 参加登録から削除
- /wclear : 参加登録リストを全削除
- /wlist : 参加登録リストを表示
- /wstart : 人狼ゲーム開始
- /wend : ゲーム強制終了

### OP不要
- /wred (message) : 人狼チャットへメッセージ送信
- /wv (player) : 投票先/能力行使先を選択

## 留意点
一緒にマイクラをやってくれる友達がいないので、動作検証ができていない部分が多大にあります。

## 対応バージョン
- Spigot 1.15.2
